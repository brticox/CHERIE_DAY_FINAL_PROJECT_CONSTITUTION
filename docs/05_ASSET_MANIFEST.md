# ASSET MANIFEST

This manifest defines the required assets for the CHERIE DAY cinematic website. The hero must be built from separate layers, not one flat AI video. `HERO_CINEMATIC_SYSTEM.md` is the governing creative and technical specification for the homepage hero; this manifest lists the asset deliverables that support it.

Hero rule: the generated hero video is a background ambience layer only. The real interactive hero is built from DOM-based transparent product layers controlled by GSAP ScrollTrigger and Lenis.

## 1. Hero Background Video

File target:

- `hero/wedding-background-loop.mp4`
- `hero/wedding-background-loop.webm`
- `hero/wedding-background-poster.jpg`

Current source:

- `assets/hero-source/wedding-garden-background-source.mp4`

Direction:

- luxury wedding atmosphere,
- warm ivory/burgundy/brass grade,
- shallow depth of field,
- slow camera motion or locked elegant composition,
- no text,
- no logo,
- no visible brand conflicts,
- no cartoon or over-AI look.

Technical:

- 8-12 second seamless loop,
- muted autoplay,
- 1920x1080 minimum,
- mobile crop-safe center,
- compressed WebM fallback,
- poster frame required.
- must not contain final text, product objects, QR animation, logo animation, or interactive layer motion.
