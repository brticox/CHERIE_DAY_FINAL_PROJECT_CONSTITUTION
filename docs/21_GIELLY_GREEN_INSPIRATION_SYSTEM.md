# GIELLY GREEN INSPIRATION SYSTEM

This file translates the attached Gielly Green analysis into CHERIE DAY-specific design inspiration. It defines what can be inspired and what must not be copied.

## 1. What We Can Use

CHERIE DAY can be inspired by:

- centered logo/header rhythm,
- left menu trigger,
- right-side account/cart utility area,
- deep burgundy and cream contrast,
- subtle gold/brass accents,
- ivory/off-white page backgrounds,
- hairline dividers,
- editorial whitespace,
- refined product rows,
- drawer cart behavior,
- smooth Lenis scroll,
- GSAP scroll reveals,
- slow parallax imagery,
- restrained luxury menu transitions,
- tactile boutique product storytelling.

## 2. What We Must Not Copy

Do not copy:

- Gielly Green's logo, monogram, brand identity, typography files, salon logic, page text, product structure, salon branch structure, exact layouts, exact hero, image assets, or Shopify/Prismic architecture as-is.

CHERIE DAY is a wedding, gift, celebration, digital, memory, and planning Maison, not a salon.

## 3. CHERIE DAY Adaptation

Header:

- left: `Menü`
- center: CHERIE DAY wordmark/monogram
- right: `Hesabım`, `Seçimlerim`, WhatsApp or `Hayalini Tasarla`
- desktop can use a slim capsule/hairline header over cinematic pages,
- header should feel integrated into the visual world, not pasted as a SaaS navbar,
- logo stays optically centered even when left/right utilities differ in width,
- scroll state can shift from transparent/cream to subtle blur/paper tint,
- avoid dense full-width navigation on the first viewport.

Menu drawer:

- deep burgundy or ivory depending on page context,
- large Turkish section labels,
- grouped Maison worlds,
- a featured `Hayalini Tasarla` panel,
- footer shortcuts for WhatsApp, Instagram, email.
- drawer should open like a private Maison menu, not a mobile app sidebar,
- use hairline separators, generous spacing, and one featured visual/object,
- menu groups: `Maison`, `Kutlama`, `Ürünler`, `Dijital`, `Hatıra`, `Rehber`, `Destek`,
- active/hover states use brass line or soft burgundy fill, never loud color blocks.

Cart drawer:

- `Seçimlerim`,
- calm item list,
- quantity editing,
- production estimate,
- proof requirement note,
- `Güvenli Ödemeye Geç`,
- no loud discount pressure.

Color:

- CHERIE DAY burgundy can sit near `#4A0E17` as inspiration, but final values must be CHERIE DAY-owned.
- cream/off-white backgrounds are appropriate.
- brass/gold is an accent only.
- charcoal text should be soft, not pure black.
- use Gielly-like restraint: large cream fields, deep burgundy moments, charcoal reading text, brass details only.
- do not let every section become burgundy; burgundy should appear as chapter depth, drawer background, CTA accent, or editorial panel.

Motion:

- Lenis global smooth scroll with reduced-motion opt-out.
- GSAP reveals for editorial sections.
- Hero remains CHERIE DAY's layered cinematic system, not Gielly's hero.
- Product rows can use horizontal drag/snap if mobile-friendly.
- Drawer transitions stay 220-350ms, calm and precise.

Commerce:

- Product rows should feel like a boutique edit.
- Product details should tell story, material, personalization, production, delivery, proof.
- Checkout should feel complete and trustworthy, not generic.
- drawer mini-cart is a key inspiration pattern and should become `SeçimlerimDrawer`.
- cart drawer opens without full page redirect.
- product cards should support quick `Seçimlerime Ekle` only for eligible products.
- quote-required products open inquiry/proof expectation instead of pretending every product is instant-buy.

## 3.1 CHERIE Maison Chrome

Use this internal pattern for header/menu/cart:

**CHERIE Maison Chrome** = the persistent interface layer that quietly floats over the cinematic world.

Rules:

- top header is visually light and centered,
- left side is discovery: `Menü`,
- center is identity: CHERIE DAY logo/mark,
- right side is action: `Hesabım`, `Seçimlerim`, optional WhatsApp/`Hayalini Tasarla`,
- header must remain readable over video through subtle blur, tint, or hairline,
- menu drawer uses burgundy/ivory contrast,
- cart drawer uses ivory/paper surfaces for clarity,
- all labels are Turkish,
- icons are minimal line icons, not decorative symbols,
- account/cart are utilities, not dominant CTAs.

Desktop header layout:

`[Menü] [Hayalini Tasarla optional]     [CHERIE DAY]     [Hesabım] [Seçimlerim]`

Mobile header layout:

`[Menu icon]     [CHERIE DAY mark]     [Account icon] [Cart icon]`

Do not copy Gielly's exact monogram, exact nav copy, exact layout measurements, salon booking logic, or typography files.

## 4. Acceptance Rules

The result is correct if it feels like:

- Gielly-level editorial commerce rhythm,
- CHERIE DAY-specific romance and ceremony,
- Turkish luxury microcopy,
- complete Turkey commerce,
- warm Maison, not salon.

The result is wrong if it looks like:

- copied salon website,
- generic Shopify clone,
- dark British salon identity,
- marketplace product grid,
- text or layout lifted from Gielly.
