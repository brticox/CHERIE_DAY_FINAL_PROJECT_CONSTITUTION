# Content Strategy

CHERIE DAY content should convert search, education, inspiration, and inquiry into one coherent Brand House experience.

## Tone Of Voice

Voice attributes:
- French-romantic Maison sensitivity,
- literary Turkish,
- luxurious but close to the heart,
- poetic when emotion matters,
- spontaneous and human in microcopy,
- bridal editorial,
- warm Turkish,
- not cute,
- not childish,

Avoid:
- marketplace language,
- provider comparison language,
- generic wedding-site clichés,
- hard-sell discount copy,
- overexplaining operational sourcing.

## Public Language Rule

All public customer-facing copy must be Turkish.

This includes:
- navigation,
- forms,
- cart,
- checkout,
- account,
- login/register,
- order tracking,
- payment status,
- proof approval,
- support/inquiry flows,
- emails/SMS/WhatsApp templates authored by CHERIE DAY,
- legal and consent pages.

EN/AR are future localization layers only. Do not mix English UI labels into the Turkish MVP except intentional brand/world names such as collection names.

## Trust Copy

Trust copy should focus on:

- aesthetic consistency,
- clear process,
- Turkish-global event understanding,
- atelier production,
- selected implementation network,
- client stories,
- portfolio outcomes.

## Educational Copy

Educational content should make planning less overwhelming. It should answer real Turkish search questions:
- Evde söz nasıl hazırlanır?
- Nişan masasında neler olur?
- Dijital davetiye nasıl seçilir?
- Düğün web sitesi ne işe yarar?
- Düğün bütçesi nasıl planlanır?

Each guide should end with an CHERIE DAY path, not a generic provider search.

## Turkish Rewrite Rules

- Use natural Turkish, not literal English translations.
- Use refined, literary Turkish without becoming stiff.
- Let emotional moments breathe with poetic warmth.
- Let transactional moments stay clear, reassuring, and precise.
- Use intimate but respectful phrasing; close to the heart, never childish.
- Prefer `deneyim`, `dünya`, `tasarım`, `üretim`, `koordinasyon`, `anı`, `koleksiyon`.
- Use `Düğün Web Sitesi` for clarity, even if a more poetic phrase appears in headings.
- Use English collection names intentionally as brand/world names.
- Keep admin/internal terms out of public copy.

Avoid:
- `Cart`
- `Checkout`
- `Login`
- `Register`
- `Submit`
- `Order tracking`

- `Bu parçayı gününüze uyarlamak isterseniz bize küçük bir not bırakabilirsiniz.`
