# Visual Asset Production Checklist

This checklist is general. It does not prescribe a specific hero concept.

## Required Brand Assets

- official logo SVG,
- official monogram SVG,
- official stamp/seal SVG,
- favicon/social mark exports,
- brand-safe color tokens,
- typography decisions.

## Creative Asset Intake

For any approved visual concept, collect:

- source files,
- generated images or videos,
- transparent objects if used,
- posters/placeholders,
- mobile variants,
- licensing/source notes,
- optimization targets,
- alt text guidance.

## Production Requirements

- Compress raster images to WebP/AVIF where suitable.
- Compress video to web-ready MP4/WebM where suitable.
- Provide poster/fallback images for motion assets.
- Avoid oversized initial payloads.
- Keep brand marks as vector where possible.
- Do not embed important Turkish text in non-editable imagery.

## Handoff Requirements

Every asset must include:

- intended usage,
- file path,
- dimensions,
- format,
- optimization status,
- owner/source,
- whether it is final, placeholder, or reference.
