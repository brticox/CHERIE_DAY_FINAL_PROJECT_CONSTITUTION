# Conversion Analysis

This report defines how CHERIE DAY should move users from inspiration to action without becoming a marketplace.

## Core Funnel

1. Inspiration
2. Understanding
3. Trust
4. Guided self-definition
5. Lead capture
6. Account/cart/checkout or consultation path
7. Payment, proof approval, and order tracking
8. Support, repeat purchase, and future client portal

## Inspiration To Action

Users may enter through:
- homepage,
- collections,
- Product House,
- Rehber article,
- Digital page,
- Experience page,
- Instagram/WhatsApp.

Each entry point should route toward one of:
- `Hayalini Tasarla`
- `Teklif Al`
- `Seçimlerime Ekle`
- `Güvenli Ödemeye Geç`
- `Hesabıma Git`
- `Siparişimi Takip Et`
- `Koleksiyonu İncele`
- `Dijital Davetiye Başlat`
- `WhatsApp ile yaz`

## Product & Commerce Paths

- Product discovery → product detail → personalization → cart → checkout → payment → proof/order tracking
- Collection discovery → complete the collection → cart or bespoke quote
- Digital invitation → theme preview → RSVP/digital project lead
- Gift object → personalization → cart → Turkey checkout
- Rehber → related product/collection → cart, quote, or lead

## Account And Order Paths

Account paths:
- Register/login before or during checkout.
- Account dashboard shows orders, addresses, support, and proof approvals.
- Customers can return to order details from email/WhatsApp links.

Order paths:
- Paid product order → confirmation → order detail → proof approval if needed → production → shipment → delivery.
- Failed payment → clear retry path and support CTA.
- Bespoke order → inquiry/quote path first, then payment/order once configured internally.

All public account and commerce copy must be Turkish.

## ## Lead Capture Locations

High-intent:
- hero section,
- after process explanation,
- after portfolio/client story,
- product detail,
- collection detail,
- Rehber article midpoint and end,
- footer.

Do not force lead capture before the user understands CHERIE DAY’s value.

## Hayalini Tasarla

Purpose:
Turn vague desire into structured brief.

Recommended flow:
1. Event type
2. Date or season
3. Location
4. Guest count
5. Desired mood/style
6. Selected collection or inspiration
7. Needed modules: Experience, Product House, Digital, Memory, Planning
8. Budget band
9. Contact details
10. Optional file/inspiration upload

Output:
Lead record, recommended CHERIE DAY paths, internal brief summary, WhatsApp handoff option.

## Product Inquiry

Product inquiry should feel like customization and support alongside commerce, not a marketplace seller message.

Flow:
1. Product/category
2. Collection
3. Quantity
4. Personalization
5. Event date
6. Delivery city/country
7. Contact

CTA:
`Sepete Ekle`, `Kişiselleştir`, `Ürün Hakkında Sor`, or `Ürün Teklifi Al` depending on product type.

## Quote Request

Quote request should support full experience inquiries.

Flow:
1. What are you planning?
2. Which CHERIE DAY modules do you need?
3. Date/location/guest count
4. Style/collection
5. Budget band
6. Notes and contact

Trust before CTA:
Show process, portfolio, testimonials, and FAQ before the main quote form on detail pages.

## WhatsApp Handoff

WhatsApp should be available but not the only funnel.

Use cases:
- quick questions,
- high-intent mobile users,
- post-form confirmation,
- product personalization follow-up,
- consultation scheduling.
- payment/order support if a customer needs help.

Message template:
`Merhaba CHERIE DAY, [event type] için bilgi almak istiyorum. Tarih: [date]. Konum: [location].`

## Trust Points Before CTA

Before asking for a serious inquiry, show:
- one-brand promise,
- process steps,
- portfolio proof,
- client story,
- what CHERIE DAY handles,
- FAQ,
- production/delivery clarity.

## Recommended CHERIE DAY Funnel

Homepage:
Hero -> Worlds -> Experiences -> Collections -> How it works -> Portfolio -> Hayalini Tasarla.

Experience:
Hero -> What CHERIE DAY creates -> Process -> Matching modules -> Client story -> FAQ -> Quote.

Collection:
Hero -> Mood/materials -> Products -> Digital theme -> Styling direction -> Portfolio -> Adapt this world.

Product:
Gallery -> Product details -> Matching collection -> Personalization -> Cart/checkout or Product inquiry -> Order tracking.

Rehber:
Answer -> Examples -> CHERIE DAY note -> Related module -> FAQ -> CTA.
