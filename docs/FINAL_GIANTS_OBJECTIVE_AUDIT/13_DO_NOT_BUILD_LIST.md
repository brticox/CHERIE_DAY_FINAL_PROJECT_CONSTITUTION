# Do Not Build List

Do not build these in MVP:

- public vendor profiles,
- supplier search,
- vendor comparison,
- marketplace ratings/reviews per seller,
- public saved vendors,
- multi-seller cart,
- public supplier credits,
- full self-serve wedding website builder,
- live RSVP dashboard,
- guest-list management,
- public event portal,
- supplier portal,
- international shipping,
- international storefront,
- multi-currency storefront,
- EN/AR storefront,
- discount/urgency badge system,
- heavy admin analytics,
- advanced inventory automation,
- AI copy generation inside CMS,
- full proof markup editor,
- city-directory SEO pages,
- thin programmatic SEO pages,
- flat AI-video-only hero,
- burned-in hero text,
- unoptimized source images as production assets.
