# Contradictions And Missing Decisions

## Contradictions / Tensions

1. MVP is described as both inquiry-led and full paid commerce. This is solvable, but day-one product eligibility must be fixed.
2. Digital invitations and wedding websites appear as future platform value but can read like MVP features in some strategic sections.
3. The hero is rightly ambitious but competes with practical asset readiness.
4. Luxury restraint is specified, while motif list risks overdecoration if all motifs appear together.
5. Turkey-only operations are clear, but foreign-issued card acceptance must stay carefully worded to avoid implying international commerce.
6. CMS/CRM/admin scope is broad enough to resemble a large internal product unless MVP admin screens are reduced.
