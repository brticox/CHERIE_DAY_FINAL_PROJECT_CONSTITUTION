# Opening Experience Creative Slot

This file no longer defines a specific hero design.

The opening experience is a creative slot to be directed separately by the approved creative agent.
No previous hero concept, video treatment, object sequence, scene style, or animation pattern is mandatory.

## Purpose

The opening experience must introduce CHERIE DAY with luxury, clarity, brand confidence, and conversion awareness.

It may use video, generated imagery, 2D layers, 3D, canvas, WebGL, motion, still composition, or another suitable approach if approved.

## Non-Negotiable Constraints

- Use official CHERIE DAY brand marks correctly.
- Keep all public copy in Turkish.
- Preserve readability on desktop and mobile.
- Provide reduced-motion fallback.
- Avoid scroll traps.
- Do not block access to store, inquiry, account, or checkout.
- Keep performance measurable and acceptable.
- Use real HTML text for important copy.
- Do not burn required UI text into images.
- Do not distort logo, monogram, stamp, or QR-like operational assets.

## Creative Workflow

Before implementation, the creative agent must provide:

1. concept name,
2. short creative rationale,
3. first viewport composition,
4. motion/interactivity concept,
5. asset list,
6. desktop/tablet/mobile behavior,
7. accessibility and reduced-motion fallback,
8. performance budget,
9. connection points to store and inquiry flows.

Implementation starts only after this concept is approved.

## What Is Intentionally Removed

This constitution intentionally removes:

- fixed hero object lists,
- fixed video requirements,
- fixed scroll timelines,
- fixed reference-site imitation,
- fixed visual motifs,
- fixed animation choreography.

The opening must be invented fresh for CHERIE DAY using brand assets, business goals, and approved creative direction.
